# Movie Recommendation System

This is a Flask-based Movie Recommendation System that suggests movies based on user selection.

## Features
- Movie recommendation using machine learning
- Simple web interface
- Fast predictions using pickle model

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn

## Installation

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

## Project Structure

app.py  
movie_dict.pkl  
templates/  
requirements.txt
